# Topdo FAQ

## 安装后打不开怎么办？

当前安装包未做 Apple notarization，首次打开可能会被 macOS 拦截。

可以尝试：

1. 打开 `系统设置 → 隐私与安全性`，允许打开 Topdo。
2. 或执行：

```bash
xattr -cr /Applications/Topdo.app
```

## Topdo 支持 Windows 吗？

当前只发布 macOS Universal 版本，支持 Apple Silicon 和 Intel Mac。Windows 版本暂不在当前计划内。

## 本地模式和飞书同步有什么区别？

本地模式不需要配置，数据保存在当前 Mac 上，适合快速开始。

飞书同步会把任务同步到你指定的飞书多维表格，适合多设备查看或希望保留云端记录的用户。

## 切换到本地模式会删除飞书配置吗？

不会。切换模式只改变当前使用方式，已经填写的飞书配置会保留，后续可以再切回飞书同步。

## 飞书同步为什么要配置应用凭证？

Topdo 需要通过飞书开放平台 API 读写你指定的多维表格。应用凭证用于证明这次访问来自你自己创建的飞书应用。

## 飞书同步需要哪些权限？

Topdo 只需要多维表格相关能力：读取表格、读取记录、创建记录、更新记录和删除记录。

如果测试连接提示权限不足，请检查飞书开放平台里是否给应用开通了多维表格相关权限，并确认应用已发布。

## 为什么收不到通知？

请检查：

1. macOS 是否允许 Topdo 发送通知。
2. 任务或习惯是否设置了提醒时间。
3. 任务是否已经完成，已完成任务不会继续提醒。
4. macOS 通知中心是否把通知折叠到了历史通知里。

## 任务只选日期不选时间，会什么时候到期？

只选日期时，Topdo 默认按当天 `23:59` 作为到期时间。

## 重复任务会自动生成下一次吗？

会。重复任务完成后，Topdo 会根据重复规则生成下一次任务。例如每周二重复的任务，本周完成后会生成下一周二的任务。

## 支持作者会影响功能使用吗？

不会。Topdo 核心功能保持免费可用，支持作者完全自愿。
