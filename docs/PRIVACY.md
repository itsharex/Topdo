# Topdo 隐私说明

Topdo 的设计原则是：本地优先、用户主动同步、尽量少收集。

## 数据存储

在本地模式下，任务、习惯和设置保存在你自己的 Mac 上。Topdo 当前没有自建服务器，也不会把你的任务内容上传到作者服务器。

## 飞书同步

只有当你主动切换到飞书同步模式并完成配置后，Topdo 才会访问你指定的飞书多维表格。

Topdo 会使用你填写的飞书应用凭证访问以下内容：

- 读取你指定的多维表格记录。
- 创建任务记录。
- 更新任务记录。
- 删除你在 Topdo 中删除的任务记录。

Topdo 不会扫描你的其他飞书文档、群聊或工作台内容。

## 凭证保存

你填写的 `App ID`、`App Secret`、`App Token` 和 `Table ID` 只用于连接你自己的飞书多维表格。

其中 `App Secret` 会保存在本机配置中，Topdo 页面不会再次明文展示。请不要把这些凭证发给他人。

## 通知权限

任务提醒和习惯提醒会使用 macOS 系统通知。通知内容来自你自己创建的任务或习惯。

如果你不希望 Topdo 发送通知，可以在 macOS 系统设置中关闭 Topdo 的通知权限，或在 Topdo 设置页关闭提醒能力。

## 外部链接

Topdo 中的以下入口会打开系统默认浏览器访问公开页面：

- GitHub 项目主页
- 用户反馈地址
- 飞书同步配置指南
- 隐私说明
- 支持作者页面
- 检查更新时的 GitHub Release 页面

这些页面由 GitHub 或飞书等第三方服务承载，访问时会遵循对应平台的隐私政策。

## 支持作者

支持作者完全自愿，不影响任何功能使用。赞赏码页面托管在 GitHub，仅用于用户主动支持作者。

## 联系与反馈

如果你对数据安全或隐私说明有疑问，可以通过 GitHub Issues 反馈：
