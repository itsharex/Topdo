# Topdo 飞书同步配置指南

Topdo 默认可以直接使用本地模式。如果你希望任务同步到飞书多维表格，可以按下面步骤配置飞书同步。

整个流程分为 4 步：打开模板、解析表格链接、填写应用凭证、测试连接。

## 准备工作

你需要：

- 一个飞书账号。
- 已安装 Topdo。
- 能访问飞书开放平台。

## Step 1：打开并复制 Topdo 模板

1. 打开 Topdo 设置页。
2. 切换到 `飞书同步`。
3. 点击 `打开 Topdo 模板`。
4. 在飞书中把模板复制到自己的空间。
5. 打开复制后的多维表格。

建议使用模板，不要手动新建表格。模板已经包含 Topdo 需要的字段。

## Step 2：复制多维表格链接并解析

1. 打开你刚复制的多维表格。
2. 复制浏览器地址栏里的完整链接。
3. 回到 Topdo 设置页。
4. 展开 `多维表格信息`。
5. 把链接粘贴到 `多维表格链接`。
6. 点击 `解析`。

Topdo 会自动提取：

- `App Token`：多维表格链接中 `/base/` 后面的 token。
- `Table ID`：链接参数中的 `table=...`。

你通常不需要手动填写这两个字段。

## Step 3：创建飞书应用并填写凭证

Topdo 需要通过你自己的飞书应用访问多维表格。

1. 打开 [飞书开放平台](https://open.feishu.cn/app)。
2. 创建一个企业自建应用，例如命名为 `Topdo 同步`。
3. 进入应用的 `凭证与基础信息`。
4. 复制 `App ID` 和 `App Secret`。
5. 回到 Topdo 设置页。
6. 展开 `飞书应用凭证`。
7. 填写 `App ID` 和 `App Secret`。

`App Secret` 保存后会存放在本机配置中，页面不会再次明文展示。

## Step 4：开通权限并测试连接

在飞书开放平台中，为你的应用开通多维表格相关权限。

Topdo 需要的权限范围是：

- 读取多维表格
- 读取记录
- 创建记录
- 更新记录
- 删除记录

开通权限后，请发布应用。企业账号可能需要管理员审批。

回到 Topdo 设置页：

1. 点击 `测试连接`。
2. 看到连接成功后，点击 `保存设置`。
3. 回到任务页，新建一个测试任务。
4. 打开飞书多维表格，确认任务已同步。

## 常见问题

### 测试连接提示权限不足

检查应用是否已开通多维表格相关权限，并确认应用已经发布。企业账号可能还需要管理员审批。

### 提示 App Token 或 Table ID 不正确

重新打开复制后的多维表格，复制完整浏览器链接，再回到 Topdo 中重新解析。

### App Secret 输入框为空，是不是丢了？

不是。保存后 Topdo 不会在页面明文展示 App Secret。如果你没有更换 Secret，可以留空保存。

### 换电脑后怎么同步？

在新电脑安装 Topdo，重新填写同一组 App ID、App Secret 和多维表格链接即可。

### 本地模式会删除飞书数据吗？

不会。切换到本地模式只改变当前使用方式，不会删除飞书多维表格里的数据，也不会清空已保存的飞书配置。

## 数据安全说明

Topdo 只读写你在设置页指定的那张多维表格。更多说明见 [隐私说明](PRIVACY.md)。
