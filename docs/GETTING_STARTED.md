# Topdo 快速上手

Topdo 是一个 macOS 桌面待办工具，适合把任务、习惯和提醒放在屏幕上，减少“写了但忘记看”的情况。

## 1. 安装 Topdo

1. 下载最新版 `Topdo_2.0.2_universal.dmg`。
2. 打开 DMG，把 `Topdo.app` 拖到 `Applications`。
3. 首次打开如果被 macOS 拦截，在系统设置里允许打开，或执行：

```bash
xattr -cr /Applications/Topdo.app
```

## 2. 选择使用模式

Topdo 支持两种模式：

| 模式 | 适合场景 | 是否需要配置 |
|---|---|---|
| 本地模式 | 先快速记录和管理自己的任务 | 不需要 |
| 飞书同步 | 希望任务同步到飞书多维表格 | 需要配置飞书应用和表格 |

第一次使用建议先选择本地模式。确认 Topdo 适合自己的工作流后，再按 [飞书同步配置指南](FEISHU_SETUP.md) 接入飞书同步。

## 3. 创建第一个任务

1. 点击右上角 `+`，或按 `⌘N`。
2. 输入任务名称。
3. 按需设置优先级、截止日期、提醒、重复和子任务。
4. 点击 `创建`。

常见任务示例：

- 紧急：今天 18:00 前确认活动素材。
- 重要：本周五前完成产品复盘。
- 普通：阅读一篇行业文章。
- 重复：每周一整理周报计划。

## 4. 推进任务状态

任务支持三种状态：

- 待办：还没开始。
- 进行中：正在处理。
- 已完成：已经完成。

点击任务左侧圆圈可以推进状态，也可以在任务详情里修改。

## 5. 设置提醒

任务支持到期提醒和提前提醒。提醒会通过 macOS 系统通知触发；当 Topdo 正在前台时，也会显示应用内提醒。

如果没有收到通知，请检查：

1. macOS 系统设置里是否允许 Topdo 发送通知。
2. 任务是否设置了截止时间。
3. 任务是否已经完成，已完成任务不会继续提醒。

## 6. 使用习惯模块

如果你有每天或每周固定要做的事情，可以在设置页开启习惯模块。

适合放进习惯的事情：

- 喝水
- 阅读
- 运动
- 每日复盘

习惯和任务是两个入口：任务更适合有明确完成目标的事项，习惯更适合长期重复的行为。

## 7. 常用快捷键

- `⌘N`：新建任务
- `⌘F`：搜索任务
- `⌘K`：打开快捷键面板
- `⌘J`：切换任务 / 习惯
- `⌘1 ~ ⌘4`：切换任务筛选
